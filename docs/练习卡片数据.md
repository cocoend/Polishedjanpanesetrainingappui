# 初級（10枚）

## 1. ペンを説明する【PREP】

- **説明の目標**：ペンが何か、何に便利かを簡単に説明する
- **使うと良いキーワード**：文房具、書く、インク、メモ、便利
- **役立つ表現**：これは〜です。理由は、〜できるからです。例えば、〜のときに使います。つまり、〜に便利です。
- **ヒント**：最初に「何か」を言う → 次に使い道 → 具体例を1つ → 最後に便利さでまとめる

## 2. スマホを説明する【PREP】

- **説明の目標**：スマホの特徴と使う場面を短く説明する
- **使うと良いキーワード**：連絡、アプリ、調べる、写真、持ち歩く
- **役立つ表現**：スマホは〜です。理由は、〜ができるからです。例えば、〜するときに使います。そのため、とても便利です。
- **ヒント**：機能を全部言わなくていいです。大事な特徴を2つくらい選ぶと話しやすいです。

## 3. かさを説明する【PREP】

- **説明の目標**：かさの役割と使うタイミングを説明する
- **使うと良いキーワード**：雨、ぬれる、防ぐ、持つ、天気
- **役立つ表現**：かさは〜です。理由は、〜を防げるからです。例えば、〜の日に使います。だから、あると安心です。
- **ヒント**：まず「何のための物か」を言うと、説明がわかりやすくなります。

## 4. 冷蔵庫を説明する【PREP】

- **説明の目標**：冷蔵庫の基本機能をやさしく説明する
- **使うと良いキーワード**：食べ物、飲み物、冷やす、保存、新鮮
- **役立つ表現**：冷蔵庫は〜です。理由は、〜を冷たく保存できるからです。例えば、〜を入れます。このように、生活でよく使います。
- **ヒント**：機能 → 何を入れるか → 便利な点、の順で話すと自然です。

## 5. コンビニを説明する【PREP】

- **説明の目標**：コンビニがどんな店かを簡単に説明する
- **使うと良いキーワード**：店、食べ物、飲み物、24時間、便利
- **役立つ表現**：コンビニは〜です。理由は、いろいろな物を買えるからです。例えば、〜や〜があります。そのため、とても便利なお店です。
- **ヒント**：店の説明では、「何があるか」と「なぜ便利か」を入れると話しやすいです。

## 6. 電子レンジの使い方を説明する【ステップ解説法】

- **説明の目標**：電子レンジの基本的な使い方を順番に説明する
- **使うと良いキーワード**：入れる、時間、ボタン、あたためる、取り出す
- **役立つ表現**：まず、食べ物を入れます。次に、時間を設定します。そのあと、スタートボタンを押します。最後に、温まったら取り出します。
- **ヒント**：1文に1動作を入れるとわかりやすいです。順番を飛ばさないようにしましょう。

## 7. 電車の乗り方を説明する【ステップ解説法】

- **説明の目標**：駅で電車に乗る流れを説明する
- **使うと良いキーワード**：切符、ICカード、改札、ホーム、乗る
- **役立つ表現**：まず、切符を買うかICカードを使います。次に、改札を通ります。そのあと、ホームで電車を待ちます。最後に、電車に乗ります。
- **ヒント**：場所の流れも意識すると話しやすいです。駅 → 改札 → ホーム → 電車、の順です。

## 8. インスタントラーメンの作り方を説明する【ステップ解説法】

- **説明の目標**：簡単な料理手順を順番に説明する
- **使うと良いキーワード**：お湯、入れる、待つ、スープ、混ぜる
- **役立つ表現**：まず、お湯を入れます。次に、数分待ちます。そのあと、スープを入れて混ぜます。最後に、できあがりです。
- **ヒント**：手順説明では、細かすぎず、必要な動作だけを言うのがコツです。

## 9. 洗濯機の使い方を説明する【ステップ解説法】

- **説明の目標**：洗濯機の基本操作を説明する
- **使うと良いキーワード**：服、入れる、洗剤、スタート、干す
- **役立つ表現**：まず、服を洗濯機に入れます。次に、洗剤を入れます。そのあと、ボタンを押してスタートします。最後に、終わったら服を干します。
- **ヒント**：最後に「どうなるか」を入れると、説明が終わった感じが出ます。

## 10. コンビニで荷物を送る方法を説明する【ステップ解説法】

- **説明の目標**：荷物を送る流れをやさしく説明する
- **使うと良いキーワード**：荷物、伝票、店員、料金、送る
- **役立つ表現**：まず、荷物を準備します。次に、伝票を書きます。そのあと、店員さんに渡します。最後に、料金を払って発送します。
- **ヒント**：手順だけでなく、必要な物も少し入れると、より実用的な説明になります。

# 中級（10枚）

## 1. 水筒を説明する【PREP】

- **説明の目標**：水筒の特徴とメリットを説明する
- **使うと良いキーワード**：飲み物、持ち歩く、保温、保冷、便利
- **役立つ表現**：水筒は〜です。理由は、飲み物の温度を保てるからです。例えば、〜のときに使います。そのため、外出のときに便利です。
- **ヒント**：初級より少し詳しく、「どんな場面で便利か」まで入れてみましょう。

## 2. キャッシュレス決済を説明する【PREP】

- **説明の目標**：キャッシュレス決済の意味と良さを説明する
- **使うと良いキーワード**：現金、スマホ、カード、支払う、早い
- **役立つ表現**：キャッシュレス決済は〜です。理由は、現金がなくても支払えるからです。例えば、〜で使えます。つまり、会計を早くできる方法です。
- **ヒント**：少し抽象的な物を説明するときは、「何か」→「どう便利か」で整理すると話しやすいです。

## 3. オンライン会議アプリを説明する【PREP】

- **説明の目標**：オンライン会議アプリの役割を説明する
- **使うと良いキーワード**：会議、話す、画面、共有、仕事
- **役立つ表現**：オンライン会議アプリは〜です。理由は、離れた場所でも話せるからです。例えば、〜のときに使います。そのため、仕事でよく使われます。
- **ヒント**：機能をたくさん言うより、「何のためのアプリか」をはっきりさせるのが大切です。

## 4. ATMでお金を入金する手順を説明する【ステップ解説法】

- **説明の目標**：ATMの入金手順を順番に説明する
- **使うと良いキーワード**：カード、暗証番号、入金、現金、確認
- **役立つ表現**：まず、カードを入れます。次に、暗証番号を入力します。そのあと、入金を選んで現金を入れます。最後に、金額を確認します。
- **ヒント**：ボタン操作と確認の流れを入れると、説明が実際っぽくなります。

## 5. スマホでアラームを設定する方法を説明する【ステップ解説法】

- **説明の目標**：スマホ操作を相手にわかるように説明する
- **使うと良いキーワード**：時計アプリ、時間、設定、保存、鳴る
- **役立つ表現**：まず、時計アプリを開きます。次に、アラームの画面を選びます。そのあと、時間を設定して保存します。最後に、指定した時間にアラームが鳴ります。
- **ヒント**：アプリ名や画面名を入れると、聞く人がイメージしやすくなります。

## 6. レストランを予約する流れを説明する【ステップ解説法】

- **説明の目標**：予約の手順を自然に説明する
- **使うと良いキーワード**：電話、人数、時間、名前、確認
- **役立つ表現**：まず、お店に電話します。次に、人数と時間を伝えます。そのあと、名前を伝えて予約内容を確認します。最後に、予約完了です。
- **ヒント**：相手に何を伝えるかも入れると、ただの順番説明より一段わかりやすくなります。

## 7. 電車とバスの違いを説明する【PREP】

- **説明の目標**：2つのものを比べて説明する
- **使うと良いキーワード**：時間、料金、便利、混む、駅
- **役立つ表現**：私は、〜のほうが〜だと思います。理由は、〜からです。例えば、電車は〜ですが、バスは〜です。そのため、場合によって使い分けます。
- **ヒント**：比較では、最初に自分の結論を言うと、話がぶれにくくなります。

## 8. スーパーとコンビニの違いを説明する【PREP】

- **説明の目標**：店の特徴を比較して説明する
- **使うと良いキーワード**：値段、種類、近い、便利、買い物
- **役立つ表現**：スーパーとコンビニは似ていますが、違いもあります。理由は、〜です。例えば、スーパーは〜で、コンビニは〜です。つまり、目的によって選ぶとよいです。
- **ヒント**：比較では、「共通点を少し → 違いをはっきり」がわかりやすいです。

## 9. ノートとメモアプリの違いを説明する【PREP】

- **説明の目標**：アナログとデジタルの違いを整理して話す
- **使うと良いキーワード**：書く、保存、検索、手軽、使いやすい
- **役立つ表現**：私は、〜には〜が向いていると思います。理由は、〜からです。例えば、ノートは〜しやすく、メモアプリは〜しやすいです。そのため、使う場面で選べます。
- **ヒント**：どちらが良いかを決めなくても大丈夫です。「場面によって違う」とまとめても自然です。

## 10. 在宅勤務と出社勤務の違いを説明する【PREP】

- **説明の目標**：働き方の違いを整理して説明する
- **使うと良いキーワード**：通勤、集中、コミュニケーション、自由、効率
- **役立つ表現**：在宅勤務と出社勤務には、それぞれ特徴があります。理由は、働く環境が違うからです。例えば、在宅勤務は〜ですが、出社勤務は〜です。つまり、自分の仕事に合う方を選ぶことが大切です。
- **ヒント**：比較するときは、1つの基準で比べると話しやすいです。たとえば「時間」「集中」「連絡」などです。

# 上級（10枚）

## 1. 自己紹介をする【PREP】

- **説明の目標**：面接で自分の経歴をわかりやすく伝える
- **使うと良いキーワード**：経歴、経験、担当、強み、現在
- **役立つ表現**：私は〜を経験してきました。理由は、主に〜の業務を担当してきたからです。例えば、〜に携わりました。この経験を今後も活かしたいです。
- **ヒント**：最初に「どんな人か」を一言で言うと、その後が聞きやすくなります。

## 2. 転職理由を説明する【PREP】

- **説明の目標**：前向きな転職理由を整理して伝える
- **使うと良いキーワード**：成長、挑戦、開発、経験、今後
- **役立つ表現**：私が転職を考えた理由は〜です。理由は、〜に挑戦したいからです。例えば、現在は〜を担当していますが、今後は〜を伸ばしたいと考えています。そのため、転職を決めました。
- **ヒント**：不満を中心にせず、「これから何をしたいか」を中心に話すと印象が良いです。

## 3. 志望動機を説明する【PREP】

- **説明の目標**：応募先を選んだ理由を論理的に話す
- **使うと良いキーワード**：魅力、事業、成長環境、活かす、貢献
- **役立つ表現**：御社を志望する理由は〜です。理由は、〜に魅力を感じたからです。例えば、私の〜の経験は、御社の〜で活かせると考えています。そのため、入社後は〜の形で貢献したいです。
- **ヒント**：会社の魅力だけで終わらず、「自分がどう活かせるか」まで言うのがポイントです。

## 4. 現在の担当業務を説明する【PREP】

- **説明の目標**：今の仕事を相手が理解しやすい形で説明する
- **使うと良いキーワード**：担当、業務内容、テスト、改修、対応
- **役立つ表現**：現在、私は〜を担当しています。主な理由は、〜の業務に関わっているからです。例えば、〜や〜を行っています。この経験を通して、〜の力を身につけました。
- **ヒント**：仕事説明では、「何をしているか」と「何ができるようになったか」をセットで話すと強いです。

## 5. 作業の流れを説明する【ステップ解説法】

- **説明の目標**：業務手順を順番に説明する
- **使うと良いキーワード**：確認、実施、記録、共有、完了
- **役立つ表現**：まず、依頼内容を確認します。次に、必要な作業を実施します。そのあと、結果を記録して関係者に共有します。最後に、問題がなければ完了です。
- **ヒント**：仕事の手順は、「確認 → 実施 → 共有 → 完了」で整理すると伝わりやすいです。

## 6. 不具合の状況を報告する【SCQA】

- **説明の目標**：問題発生時に状況を整理して報告する
- **使うと良いキーワード**：状況、不具合、原因、影響、対応
- **役立つ表現**：現在、〜という状況です。しかし、〜という不具合が発生しています。そこで、原因の切り分けが必要です。対応として、まず〜を確認しています。
- **ヒント**：SCQAでは、感想より事実を先に言います。状況 → 問題 → 必要な対応 → 今やっていること、の順です。

## 7. 作業遅延について説明する【SCQA】

- **説明の目標**：遅れの理由と今後の対応を冷静に伝える
- **使うと良いキーワード**：進捗、遅れ、原因、影響、リカバリー
- **役立つ表現**：現在、作業は〜まで進んでいます。ただ、〜の理由で遅れが出ています。そこで、優先順位の見直しが必要です。今後は、〜して遅れを取り戻す予定です。
- **ヒント**：遅れの説明では、言い訳だけで終わらず、必ず対応策まで入れましょう。

## 8. ミスの再発防止を説明する【SCQA】

- **説明の目標**：ミスの内容と再発防止策を整理して話す
- **使うと良いキーワード**：ミス、確認不足、原因、防止、改善
- **役立つ表現**：今回、〜という状況がありました。問題は、〜というミスが起きたことです。そこで、原因を整理する必要がありました。今後は、〜を行い、再発防止につなげます。
- **ヒント**：SCQAでは、「何が起きたか」だけでなく、「次にどう防ぐか」がとても大切です。

## 9. 引き継ぎ手順を説明する【ステップ解説法】

- **説明の目標**：後任者に仕事の進め方を順番に伝える
- **使うと良いキーワード**：確認、資料、連絡先、注意点、共有
- **役立つ表現**：まず、案件の資料を確認します。次に、関係者と連絡先を把握します。そのあと、日々の作業内容を確認します。最後に、注意点を共有して引き継ぎ完了です。
- **ヒント**：引き継ぎ説明では、「何を見るか」「誰と関わるか」「何に注意するか」を入れると実務的です。

## 10. 業務改善案を提案する【SCQA】

- **説明の目標**：現状の課題と改善案を論理的に説明する
- **使うと良いキーワード**：現状、課題、効率、改善、自動化
- **役立つ表現**：現在、〜という状況です。しかし、〜という課題があります。そこで、〜を改善する必要があります。提案として、〜を導入すると効率化できると考えています。
- **ヒント**：改善提案は、いきなり案を言うより、現状と課題を先に示すと説得力が上がります。
